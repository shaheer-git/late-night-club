| Layer     | Tech |
|-----------|------|
| Mobile    | React Native + Expo (Android / iOS / Web) |

| Backend   | FastAPI + PostgreSQL + PostGIS |
| Cache     | Redis |
| Jobs      | Celery |
| Auth      | JWT |
| Storage   | Cloudinary (free tier) |

## Quick Start (Full Stack)

### 1. Clone & configure

```bash
git clone https://github.com/yourorg/late-night-club
cd late-night-club
cp .env.example backend/.env
```

### 2. Start everything with Docker

```bash
docker compose up
```

This starts:
- PostgreSQL + PostGIS on port 5432
- Redis on port 6379
- FastAPI backend on http://localhost:8000
- Celery worker + beat scheduler

### 3. Run DB migrations

```bash
docker compose exec backend alembic upgrade head
```

### 4. API docs

Open http://localhost:8000/docs (Swagger UI auto-generated by FastAPI)

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/register` | Register |
| POST | `/api/auth/login` | Login тЖТ JWT tokens |
| POST | `/api/auth/refresh` | Refresh access token |
| GET  | `/api/places?lat=&lng=` | Nearby places |
| GET  | `/api/places/search?q=` | Search places |
| GET  | `/api/places/{id}` | Place detail |
| POST | `/api/places` | Add place (auth) |
| POST | `/api/verifications` | Verify status (auth) |
| GET  | `/api/users/me` | My profile |
| GET  | `/api/cities` | All cities |

## Environment Variables

Copy `.env.example` тЖТ `backend/.env` and fill in:

- `R2_*` тАФ Cloudflare R2 for image uploads (optional for MVP)
